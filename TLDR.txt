TLDR - Vocabulary Quiz Quick Reference
=======================================

BASIC USAGE
-----------
.\vocabularyquiz.ps1                                          # All topics, random order, wordlist.txt
.\vocabularyquiz.ps1 -File spanish.txt                        # Use custom file
.\vocabularyquiz.ps1 -Topic Animals                           # Specific topic
.\vocabularyquiz.ps1 -Topic "Animals Colors"                  # Multiple topics
.\vocabularyquiz.ps1 -Order List                              # Sequential order
.\vocabularyquiz.ps1 -File vocab.txt -Topic Animals -Order List  # All options
.\vocabularyquiz.ps1 -Help                                    # Show help


SETUP FILE FORMAT (wordlist.txt or custom file)
-----------------------------------------------
[Animals]
dog: a four-legged pet
cat: feline animal

[Colors]
red: color of fire
blue: color of sky

QUICK EXAMPLES
--------------
Start with default wordlist.txt file:
  .\vocabularyquiz.ps1

Use custom file with Animals topic:
  .\vocabularyquiz.ps1 -File spanish.txt -Topic Animals

Use custom file with multiple topics in list order:
  .\vocabularyquiz.ps1 -File vocab.txt -Topic "Animals Colors" -Order List

During quiz, type 'help' for command reference

Type 'topic Colors' to switch to Colors topic mid-quiz
